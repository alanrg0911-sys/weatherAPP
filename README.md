-What is weatherAPP?

It's a simple CLI app where you can look for the weather conditions of any city in the world with the help of an API(WeatherAPI). Also, you can retrieve the weather information at current location using your IP address with help of the API "IP-API"(you might need to be connected to internet in order to use this function).
And there's a simple history viewer that shows you your last searches.

You need an API key from the API WeatherBit to access to the functions of this app.

This was a simple project that I wanted to develop to learn how to use APIs.
